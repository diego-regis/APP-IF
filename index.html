📐 Modelo Hipsométrico — Validação em Campo
Como funciona: O sistema ajusta a equação de Curtis (ln(Ht) = β₀ + β₁/DAP) aos seus dados históricos. Em campo, cada medição de altura é comparada ao IC 95% do modelo — o sistema alerta quando a medição está fora do intervalo esperado para aquele DAP.
